## Tentang Sekolahku

Sekolahku adalah aplikasi manajemen sekolah berbasis website yang di bangun dan di kembangkan dengan Framework Laravel. Fitur-fitur pada aplikasi Sekolahku antara lain :

- Website Sekolah
- PPDB
- Perpustakaan
- Sistem Pembayaran Sekolah SPP 
- Alumni
